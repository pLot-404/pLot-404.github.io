# 山田事件也　PWAアプリ

R3年度桐陰祭企画「山田事件也」のPWA移植&続編。
スマホ向けUIを整備するのは最重要課題。

__重要：mainブランチを公開用とし、普段の開発はdevブランチで行うこと。__

## /

ホームページ。UI面を一新する予定。


## chat

チャットページ。見た目を更に見やすく、特にスマホで使いやすいよう。

## map

マップページ。UnityのWebGLを用いることも視野に入れて、よりユーザーフレンドリーにする。


## コース用ページ
### court.onion

「囚人のジレンマ」編で使う。UI整備、Push通知の代替探しなどが必要。
